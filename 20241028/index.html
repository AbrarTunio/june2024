JavaScript Practical Quiz

Complete the following tasks to demonstrate your JavaScript knowledge.

Task 1: Variable Assignment and Arithmetic Operators

Create a simple calculator:

  1. Create two variables, num1 and num2, and assign them numeric values.
  2. Create a variable sum to hold the result of adding num1 and num2.
  3. Create a variable product to hold the result of multiplying num1 and num2.
  4. Log the results to the console in the specified format.

Task 2: Dynamic HTML Content Update

Display user input in the HTML document:

  1. Create an HTML structure with an input box and a button.
  2. Use JavaScript to display the input value in a paragraph when the button is clicked.

Task 3: Simple Arithmetic Operations and HTML Insertion

Perform arithmetic operations based on user input:

  1. Create an HTML structure with two input fields and a select dropdown for choosing an arithmetic operation.
  2. Write a JavaScript function to perform the operation and display the result in a paragraph.
